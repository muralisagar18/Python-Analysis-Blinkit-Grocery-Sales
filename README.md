# BlinkIT Grocery Sales Analysis 📊

## Project Overview

This is an End-to-End Data Analytics Project analyzing BlinkIT Grocery Sales using **SQL, Python, and Power BI**.

The project focuses on cleaning data, analyzing sales performance, and building an interactive dashboard for business insights.

---

## Tools & Technologies Used

* Power BI
* SQL Server
* Python
* Pandas
* Matplotlib
* Excel

---

## Project Workflow

### 1. Data Collection

* Imported dataset from Excel
* Loaded data into SQL Server

### 2. SQL Data Analysis

* Data Cleaning
* Aggregations
* Sales Analysis
* Category Performance

Example SQL Queries:

* Total Sales
* Average Sales
* Outlet Performance
* Category Analysis

---

### 3. Python Data Analysis

* Data Cleaning using Pandas
* Exploratory Data Analysis (EDA)
* Visualization using Matplotlib & Seaborn
* Sales Trend Analysis

---

### 4. Power BI Dashboard

Created Interactive Dashboard with:

KPIs:

* Total Sales
* Average Sales
* Number of Items
* Average Ratings

Visualizations:

* Sales by Outlet
* Sales by Category
* Sales Trend
* Outlet Size Analysis

Filters:

* Outlet Location
* Item Type
* Outlet Size

## Key Insights

* Tier 3 outlets generated highest sales
* Regular fat content items performed better
* Supermarket Type 1 has highest sales
* Fruits and Vegetables are top-selling category
